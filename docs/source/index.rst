Linux系统教程 
===========================
`Linux`_ 是当前互联网世界中非常重要的组成部分，在网络应用服务器、路由器、个人PC、\
手机、物联网设备等各种用途的设备中都能见到它的身影，是一个稳定、可靠、开源且免费的\
操作系统。

.. _Linux: https://baike.baidu.com/item/Linux/27050

Linux拥有众多的发行版本，其中 `RedHat`_ 目前是较为著名的服务器操作系统发行版之一，\
但其订阅收费制对多数企业来说，成本仍较为高昂。不过在开源协议的框架下，基于RedHat Linux\
源码进行构建并进行发行且免费的Linux发行版就成为了众多企业的不二之选，其中之一便是CentOS。

.. _RedHat: https://www.redhat.com/en/technologies/linux-platforms/enterprise-linux

CentOS曾是著名的RedHat Linux下游社区发行版，但其后 `改变更新发行`_ 策略，成为RedHat Linux\
的测试品。众人不得不寻找其它替代品，不过好在CentOS项目的联合创建人之一 **Gregory Kurtzer** \
又创建了 `Rocky Linux`_ 用来替代CentOS。

.. _改变更新发行: https://baike.baidu.com/item/CentOS/498948
.. _Rocky Linux: https://rockylinux.org/

本教程使用Rocky Linux进行教学，RedHat Linux有着丰富的技术文档，同时系统也非常稳定性。因此它的下游\
版本Rocky Linux可以直接使用这些丰富的技术文档，同时继承它的稳定性。

在熟悉了某一个发行版的Linux之后，其它版本的Linux也可轻松驾驭。
